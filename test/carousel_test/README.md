# ディレクトリの説明
## 概要
- カルーセル機能のみ抽出したファイル群です。
- カルーセル機能のみをテストしたい場合に使用して下さい。

## 各ファイルの説明
- line_bot_tiny.php
	- Line Bot開発用SDK
- webhook.php
	- POSTリクエストを送るファイル
- carousel_test.php
	- 定型文のカルーセル2つを生成するファイル
